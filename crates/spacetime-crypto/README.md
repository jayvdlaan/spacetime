spacetime-crypto
================

No_std-first crypto facades (RNG/entropy traits) for Spacetime.

Badges
- Docs: https://docs.rs/spacetime-crypto

Highlights
- Thin traits for RNG and entropy sources suitable for embedded/std/wasm
- Std adapters (planned): getrandom/rand_core-backed RNG

Features
- default: no_std
- alloc: optional helpers
- std: enable std-gated examples/tests
