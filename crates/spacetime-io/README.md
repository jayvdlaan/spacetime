spacetime-io
============

Minimal I/O facades oriented for no_std with std-gated examples.

Badges
- Docs: https://docs.rs/spacetime-io

Highlights
- Tiny traits and helpers for simple I/O tasks without pulling heavy deps

Features
- default: no_std
- std: enable examples and std adapters

Quickstart
- Std output example: cargo run -p spacetime-io --example std_stdout --features std
